# Traffic Thesis Unified

Unified thesis-grade pipeline for comparing:
- CNN
- LSTM
- GRU
- PINN
- PINN-LWR

Goals:
1. Same dataset for all models
2. Fixed train/val/test split
3. Metrics: MSE, MAE, RMSE, STD_ERROR
4. Lambda evaluation
5. Heavy vs stable traffic evaluation
6. Noise / attack robustness comparison
